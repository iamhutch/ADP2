==================
  USER GUIDELINES
==================
Upon installation of the House Boss app, you will be asked to enter in your zip code.  This will be used for the weather forecast function in the app. If you choose to skip it, you will need to add your zip code via the Settings section of the app if you want to use the weather feature.
